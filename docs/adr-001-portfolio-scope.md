# ADR-001: Portfolio Scope

## Status

Accepted

## Decision

This repository is a public, sanitized, educational and professional portfolio project.

## Rationale

The goal is to demonstrate architecture, engineering judgment, automation, reliability, security and cost governance without exposing proprietary work.

## Consequences

- No company code
- No secrets
- No internal hostnames
- No customer data
- No production diagrams copied from employers
